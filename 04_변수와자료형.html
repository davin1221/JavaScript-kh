변수 선언 방식

    JavaScript는 변수 선언 시 자료형이 아닌 var, let, const, (미작성) 키워드를 사용하여 선언한다. 

    - var: 변수, 변수명 중복(덮어쓰기) 가능, 함수 레벨 scope
    - let: 변수, 변수명 중복 불가, 블록 레벨 scope
    - const: 상수, 변수명 중복 불가, 블록 레벨 scope * 선언 시 초기화 필수

    사용 빈도: const > let > var(비추천)
     - js는 html 상에서 특정 요소를 선택해 지속적으로 추적하여 사용하는 경우가 많다. 
       그래서, 상수형 변수인 const에 저장하여 고정적으로 사용하는 경우가 많다. 
       * let을 쓰면 나중에 덮어쓰기 될 가능성이 있어서..

자료형

    변수 선언 시 별도의 자료형을 지정하지 않고, 변수에 대입되는 값(리터럴)에 따라 자료형이 결정된다. 
    
    * 종류
    1. string(문자열/ "",'')
    2. number(숫자/ 정수, 실수 구분X)
    3. boolean(논리값/ true, false)
    4. object(객체/ 배열(Array), JS객체{key:value}(map형태))
    5. function(함수/)
    6. undefine(자료형이 정의되지 않은 변수 = 값이 대입되지 않은 변수)
    * null: 참조하는 게 없음을 나타내는 리터럴(자료형은 아님)
