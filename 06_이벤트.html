이벤트(Event)

    1. Event: 브라우저에서의 동작, 행위 
        ex) click, keyup~press, mouseover~leave... 

    2. Event Listener: 이벤트가 발생하는 것을 대기하고 있다가 이벤트 발생 시 연결된 기능(함수) 수행 
         ex) onclick, onkeyup, onchange, onsubmit... (on이벤트)
    
    3. Event Handler: 이벤트가 리스너에 연결된 기능(함수), 이벤트가 발생했을 때 수행하고자 하는 내용을 작성하는 함수

Inline Event Model

    요소 내부에 Event Listener/Handler를 작성하는 방법 
    (형식) on이벤트명 = "함수명()"

고전 이벤트 모델

    요소가 가지고 있는 이벤트 속성(이벤트 리스너)에 이벤트 핸들러를 직접 연결하는 방법 
    장점(고전, 표준): 이벤트 제거가 가능하다.
      - 요소의 속성 중 on이벤트명(이벤트리스너) 속성에 함수 대신 null을 대입하여 이벤트 삭제

표준 이벤트 모델

    W3C에서 공식적으로 지정한 이벤트 모델(가장 많이 사용) 
     - 요소.addEventListener("이벤트명", 함수);
     - 고전 이벤트 모델의 단점 보완 가능(한 요소에 여러 이벤트 핸들러 설정 불가 -> 설정 가능)

실습: 배경색이 입력한 값으로 변하는 박스

실습2: 마우스를 올리면 박스가 늘어나며 배경색이 변경, 마우스를 떼면 박스 사라짐
